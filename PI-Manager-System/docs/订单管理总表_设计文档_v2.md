## 一、业务背景

订单管理总表是 PI Manager 系统的核心模块，用于替代 Excel 进行订单管理。

**核心目标：**
- 汇总所有订单相关数据
- 支持快速编辑和修改
- 导出与 Excel 模板一致的报表

---

## 二、计算参数

### 2.1 毛利率

| 参数 | 默认值 | 说明 |
|:---|:---:|:---|
| 基础毛利率 | 20% | 可自定义调节 |
| 毛利率级别 | 客户级 > 产品级 > 系统级 | 优先级 |

### 2.2 汇率

| 参数 | 说明 |
|:---|:---|
| 获取方式 | 动态获取（实时汇率 API） |
| 缓存策略 | 每日更新一次，不频繁刷新 |

**数据库设计：**
```sql
CREATE TABLE memo_records (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    entity_type VARCHAR(50),      -- 关联实体类型 (customer_product, pi_item 等)
    entity_id INTEGER,              -- 关联实体ID
    field_name VARCHAR(50),         -- 字段名 (remark, specifications 等)
    content TEXT,                   -- 内容
    created_by INT,                -- 创建人ID
    created_at DATETIME,            -- 创建时间
    updated_at DATETIME             -- 更新时间
);
```

## 五、计算公式

### 5.1 财务计算

```
合计金额 = 数量 × 单价

待收尾款 = 合计金额 - 已收款

预估美金报价 = 采购价 × 毛利率(20%) / 汇率

预估毛利率 = 报价 × 汇率 / 合计金额 × 100%

采购总金额 = 采购价 × 采购数 + 运费 + 杂费

总重量 = 箱数 × 整箱毛重
```

### 5.2 汇率获取

```
来源: 汇率 API (如 exchangerate-api.com)
缓存: 每日更新一次
更新策略: 每天凌晨自动更新，不频繁刷新
```

### 5.3 毛利率优先级

```
毛利率优先级: 客户级 > 产品级 > 系统级(20%)
```
