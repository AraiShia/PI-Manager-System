# 副图组件布局修复设计

## 1. 问题概述

**现象：** SubImageWidget 副图组件中的添加/删除按钮在程序内不显示

**原因：** 当前使用横向布局，滚动区域高度（70px）不足以容纳：
- 图片标签：60x60px
- 删除按钮：16x16px

容器高度不足导致内容被截断。

## 2. 设计方案

### 方案 B：改用纵向布局

将每张图片和删除按钮改为上下排列（纵向布局），使删除按钮不会因高度不足被截断。

### 2.1 当前布局（横向）

```
┌─────────────────────────────────────────────────────────────┐
│ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐                           │
│ │ IMG │ │ IMG │ │ IMG │ │  +  │  ← 删除按钮被截断          │
│ │ × │ │ × │ │ × │ │     │                           │
│ └─────┘ └─────┘ └─────┘ └─────┘                           │
│ [横向滚动]                                                │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 目标布局（纵向）

```
┌─────────────────────────────────────────────────────────────┐
│ ┌──────────────────────┐ ┌──────────────────────┐          │
│ │                      │ │                      │          │
│ │       IMG            │ │       IMG            │          │
│ │                      │ │                      │          │
│ │       [×]            │ │       [×]           │          │
│ └──────────────────────┘ └──────────────────────┘          │
│ ┌──────────────────────┐ ┌──────────────────────┐          │
│ │                      │ │                      │          │
│ │       +              │ │                      │          │
│ │                      │ │                      │          │
│ └──────────────────────┘ └──────────────────────┘          │
│ [横向滚动]                                                │
└─────────────────────────────────────────────────────────────┘
```

## 3. 代码改动

### 3.1 修改 _rebuild 方法

**文件：** `client/widgets/customer_product_dialog.py`

**改动：** 将图片项从水平排列改为网格布局（2列）

```python
def _rebuild(self):
    """重建图片显示 - 改为网格布局"""
    # 清空现有内容
    while self.container_layout.count():
        item = self.container_layout.takeAt(0)
        if item.widget():
            item.widget().deleteLater()
    
    # 清理线程
    for worker in self.image_workers:
        if not worker.isFinished():
            worker.quit()
    self.image_workers.clear()
    self.image_labels.clear()
    self.loading_labels.clear()
    
    # 使用 GridLayout 实现网格布局
    from PySide6.QtWidgets import QGridLayout
    
    # 创建网格容器
    grid_widget = QWidget()
    grid_layout = QGridLayout(grid_widget)
    grid_layout.setSpacing(10)
    grid_layout.setContentsMargins(0, 0, 0, 0)
    
    # 每行显示数量
    cols = 4
    
    for idx, url in enumerate(self.images):
        row = idx // cols
        col = idx % cols
        
        img_widget = QWidget()
        img_layout = QVBoxLayout(img_widget)
        img_layout.setContentsMargins(0, 0, 0, 0)
        img_layout.setSpacing(4)
        
        # 图片
        img_label = QLabel()
        img_label.setFixedSize(100, 100)
        img_label.setStyleSheet("border: 1px solid #e5e7eb; border-radius: 4px; background-color: #f3f4f6;")
        img_label.setAlignment(Qt.AlignCenter)
        img_label.setText("加载中...")
        img_label.setProperty("url", url)
        self.image_labels[url] = img_label
        self.loading_labels[url] = True
        img_layout.addWidget(img_label)
        
        # 删除按钮
        delete_btn = QPushButton("删除")
        delete_btn.setFixedHeight(28)
        delete_btn.setStyleSheet("""
            QPushButton {
                background-color: #ef4444;
                color: white;
                border: none;
                border-radius: 4px;
                font-size: 12px;
            }
            QPushButton:hover {
                background-color: #dc2626;
            }
        """)
        delete_btn.clicked.connect(lambda checked, i=idx: self.remove_image(i))
        img_layout.addWidget(delete_btn)
        
        grid_layout.addWidget(img_widget, row, col)
        self._load_image_async(url)
    
    # 添加按钮
    add_widget = QWidget()
    add_layout = QVBoxLayout(add_widget)
    add_layout.setContentsMargins(0, 0, 0, 0)
    add_layout.setSpacing(4)
    
    add_label = QLabel()
    add_label.setFixedSize(100, 100)
    add_label.setStyleSheet("""
        border: 2px dashed #d1d5db;
        border-radius: 4px;
        background-color: #f9fafb;
    """)
    add_label.setAlignment(Qt.AlignCenter)
    add_label.setText("+")
    add_label.setStyleSheet("""
        border: 2px dashed #d1d5db;
        border-radius: 4px;
        background-color: #f9fafb;
        color: #9ca3af;
        font-size: 24px;
    """)
    add_layout.addWidget(add_label)
    
    add_btn = QPushButton("添加")
    add_btn.setFixedHeight(28)
    add_btn.setStyleSheet("""
        QPushButton {
            background-color: #f3f4f6;
            color: #374151;
            border: none;
            border-radius: 4px;
            font-size: 12px;
        }
        QPushButton:hover {
            background-color: #e5e7eb;
        }
    """)
    add_btn.clicked.connect(self._add_new_image)
    add_layout.addWidget(add_btn)
    
    grid_layout.addWidget(add_widget, row, cols)  # 添加到当前行
    
    # 清空并替换容器
    while self.container_layout.count():
        self.container_layout.takeAt(0)
    
    # 修改滚动区域为垂直滚动
    self.scroll.setVerticalScrollBarPolicy(Qt.ScrollBarPolicy.ScrollBarAsNeeded)
    self.scroll.setWidgetResizable(True)
    
    self.container_layout.addWidget(grid_widget)
```

```python
        scaled_pixmap = pixmap.scaled(100, 100, Qt.KeepAspectRatio, Qt.SmoothTransformation)
```

## 6. 主图上传功能修复（补充）

### 6.1 问题概述

主图区域点击 `main_image_label` 或"上传主图"按钮无法正常工作。

### 6.2 问题分析

| 问题 | 说明 | 位置 |
|------|------|------|
| **QLabel 无法点击** | `main_image_label` 设置了手型光标，但没有实现点击事件 | Line 375-379 |
| **API 路径可能不匹配** | 客户端调用 `/images/upload`，需确认后端路由 | - |
| **上传方法需验证** | 文件上传的请求方式是否正确 | `upload_main_image()` |

### 6.3 问题代码

```python

self.main_image_label = QLabel()
self.main_image_label.setFixedSize(200, 200)
self.main_image_label.setStyleSheet("border: 2px dashed #d1d5db; background-color: #f9fafb;")
```

### 6.4 修复方案

#### 方案 1：添加鼠标点击事件（推荐）

```python
# 继承 QLabel 并重写 mousePressEvent
class ClickableLabel(QLabel):
    clicked = Signal()

    def __init__(self, parent=None):
        super().__init__(parent)

    def mousePressEvent(self, event):
        if event.button() == Qt.LeftButton:
            self.clicked.emit()
        super().mousePressEvent(event)

# 使用
self.main_image_label = ClickableLabel()
self.main_image_label.clicked.connect(self.upload_main_image)
```

#### 方案 2：使用 eventFilter

```python
# 在 init_ui 中
self.main_image_label.installEventFilter(self)
self.main_image_label.setProperty("uploadable", True)

def eventFilter(self, obj, event):
    if obj.property("uploadable") and event.type() == QEvent.MouseButtonPress:
        if event.button() == Qt.LeftButton:
            self.upload_main_image()
            return True
    return super().eventFilter(obj, event)
```

### 6.5 主图上传流程

```
用户点击 main_image_label

        ↓

        ↓

        ↓

```
