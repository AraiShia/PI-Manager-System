## 四、计算公式

### 4.1 财务计算

```
合计金额 = 数量 × 单价

待收尾款 = 合计金额 - 已收款

预估美金报价 = 采购价 × 毛利率(20%) / 汇率

预估毛利率 = 报价 × 汇率 / 合计金额 × 100%

采购总金额 = 采购价 × 采购数 + 运费 + 杂费

总重量 = 箱数 × 整箱毛重
```

### 4.2 参数配置

| 参数 | 默认值 | 来源 |
|:---|:---:|:---|
| 基础毛利率 | 20% | 系统设置 |
| 汇率 | 动态获取 | 每日更新 |
