# PI订单管理系统 - 开发进度文档

## 项目概述
PI订单管理系统是一个客户端-服务器架构的桌面应用程序，用于管理供应商、客户、产品和PI订单。

## 技术架构
- **客户端**: PySide6 (Qt6) 桌面应用
- **服务端**: FastAPI + SQLAlchemy
- **数据库**: MySQL (多数据库隔离)
- **打包工具**: PyInstaller

## 运行说明

### 后端启动
```bash
cd backend
pip install -r requirements.txt
python run.py
```

### 客户端运行
```bash
cd client
pip install -r requirements.txt
python main.py
```

### 打包客户端
```bash
cd client
pyinstaller client.spec
```
