# PI订单管理系统 - 原型设计方案

> 版本：v1.0
> 日期：2026-04-29

---

## 一、设计规范

### 1.1 整体风格
- **风格**：简洁、专业的B2B外贸管理系统
- **配色**：
  - 主色：#2563EB（蓝色，专业感）
  - 辅助色：#64748B（灰蓝）
  - 警示色：#F59E0B（橙色）/ #EF4444（红色）
  - 背景色：#F8FAFC（浅灰白）
  - 成功色：#10B981（绿色）

### 1.2 字体规范
- 主字体：思源黑体 / Noto Sans SC
- 标题：16-20px / Bold
- 正文：14px / Regular
- 辅助文字：12px / Regular

### 1.3 间距规范
- 页面边距：24px
- 卡片间距：16px
- 内部间距：12px
- 按钮内边距：8px 16px

### 1.4 组件库
- 基础组件：按钮、输入框、下拉框、复选框、日期选择器
- 业务组件：数据表格、表单、卡片、标签、步骤条
- 状态：默认、hover、active、disabled、loading

## 四、关键交互说明

### 4.1 新建PI时的智能填入

```
1. 选择客户
   ↓ 自动带出客户特殊要求（红色提示框）
   
2. 添加产品 → 输入OE号或搜索
   ↓ 自动查询该客户上一次购买该产品的价格和备注
   ↓ 自动填入（可修改）
   
3. 配置付款阶段
   ↓ 可添加任意数量的尾款阶段
   ↓ 自动计算各阶段金额
   
4. 确认 → 生成PI号 → 自动通知采购部门
```

### 4.2 出货时的库存筛选

```
1. 选择PI单
   ↓ 自动筛选该PI下所有未出完的库存批次
   ↓ 只显示"可出数量 > 0"的批次
   
2. 选择批次 + 输入本次出货数量
   ↓ 自动校验不超过可出数量
   ↓ 自动计算出货箱数和体积
   
3. 确认出货
   ↓ 自动扣减 pending_quantity
   ↓ 自动增加 shipped_quantity
   ↓ 自动记录库存流水
   ↓ 自动更新货物位置（在途/客户处）
```

### 4.3 版本管理提示

```
当用户修改PI的价格或备注时：
┌─────────────────────────────────────────────────┐
│  ⚠️ 检测到价格/备注变更                           │
│                                                 │
│  是否保存为新版本？                             │
│  • 保存为新版本（推荐）→ 保留历史记录            │
│  • 直接覆盖 → 不保留历史                         │
│                                                 │
│                        [取消] [保存版本]          │
└─────────────────────────────────────────────────┘
```
